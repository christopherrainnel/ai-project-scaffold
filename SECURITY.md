# Security Policy

## Supported Versions

Security fixes are applied to the `main` branch.

| Version | Supported |
|---------|-----------|
| `main`  | Yes |

## Reporting a Vulnerability

Please report suspected vulnerabilities privately.

Preferred channel:
- Use GitHub's private vulnerability reporting for this repository (Security tab).

Fallback channel:
- Open a private communication with the maintainer before public disclosure.

Please include:
- A clear description of the issue and impact.
- Reproduction steps or a minimal proof of concept.
- Affected files/paths and suggested remediation (if known).

Expected response targets:
- Initial acknowledgment: within 72 hours.
- Triage decision: within 7 days.
- Patch or mitigation timeline: shared after triage.

Do not post unpatched vulnerabilities in public issues.
