# Agent Rules (Read First)

If you are an AI agent or IDE assistant:

1) Read and follow `ops/AI_WORKFLOW.md` before doing any work.
2) Follow `.github/copilot-instructions.md` if using VS Code Copilot.

Do not:
- Overwrite governance files without showing a diff and receiving approval
- Modify `.env` (local only). Only update `.env.example`.
- Introduce dependencies without justification, version pinning, and lockfile updates

Learning preference:
- Explain JavaScript and web concepts in plain English using Python/Data Science analogies when helpful.
